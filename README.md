# Netflix Movie Performance Analytics

Built an end-to-end Netflix movie analytics project using Python and Power BI to uncover content trends, genre performance, rating patterns, and language distribution.

## Tools Used
- Python
- Pandas
- Jupyter Notebook
- Power BI
- Excel

## Project Overview
This project analyzes a Netflix movie dataset containing around 9,800+ records to extract insights related to genre distribution, ratings, release trends, and content performance.

## Key Tasks
- Data cleaning and preprocessing
- Exploratory Data Analysis (EDA)
- Trend analysis
- Rating distribution analysis
- Genre analysis
- Language analysis
- Interactive dashboard creation

## Dashboard Insights
- Total Movies
- Average Rating
- Total Votes
- Top Genres by Content Count
- Top Rated Movies
- Release Trend Over Years
- Top 5 Original Languages
- Rating Distribution
